# Personal Portfolio Website

This project is completed as part of the Prodigy Infotech Web Development Internship - Task 04.

It is a responsive personal portfolio website built using HTML, CSS, and JavaScript. The website includes sections like Home, About, Skills, Projects, and Contact. It is designed to showcase personal details, technical skills, and projects in a clean and professional way.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Responsive portfolio layout
- Home section with introduction
- About section
- Skills section
- Projects section
- Contact form
- Smooth navigation
- Mobile-friendly design
- Simple JavaScript interactions
